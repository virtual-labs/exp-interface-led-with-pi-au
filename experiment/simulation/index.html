Instructions

  • Click on the components to start interacting it with in the connection area
  • Hover over the raspberry Pi connections to get informations about various connections pin
  • Double click on various connections points to make connections
  • Double click on the empty space in the connection area to extend and bend wire connections

How to Connect

  1. Connect the LED’s anode (longer leg, positive side) to any GPIO pin on the Raspberry Pi using a wire.
  2. Connect one end of a resistor to the LED’s anode (longer leg, positive side).
  3. Connect the LED’s cathode (shorter leg, negative side) to a GND pin on the Raspberry Pi with a wire.
  4. Enter the GPIO pin number to the code section. The code section can be opened by clicking the Code button on the right upper side of the simulation area.
